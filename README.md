# 贪吃蛇游戏 (Snake Game)

这是一个经典的贪吃蛇游戏实现，包含两个版本：

1. **图形界面版本** (`snake_game.py`) - 使用 Pygame 库
2. **终端版本** (`snake_terminal.py`) - 纯文本界面，无需额外依赖

## 安装和运行

### 图形界面版本

```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python snake_game.py
```

### 终端版本

```bash
# 直接运行（无需安装额外依赖）
python snake_terminal.py
```

## 游戏控制

### 图形界面版本
- 方向键：控制蛇的移动方向
- R：游戏结束后重新开始
- Q：退出游戏

### 终端版本
- W/A/S/D：控制蛇的移动方向
- R：游戏结束后重新开始
- Q：退出游戏

## 游戏规则

- 控制蛇吃到食物（红色方块/星号）来增加分数
- 每吃一个食物，蛇的身体会变长
- 如果蛇撞到墙壁或自己的身体，游戏结束
- 游戏难度会随着分数增加而提高（蛇移动速度加快）
